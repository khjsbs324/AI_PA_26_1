브랜드 포스터 제작

제품 홍보 포스터 제작 실습 프롬프트

초안 포스터 리뷰부터 브랜드명, 레퍼런스 분석, 패키지 이미지 생성, 최종 홍보 포스터 제작까지 단계별로 진행합니다.

1. 초안 포스터 리뷰

활용 기법: 비평 프롬프트 + 체크리스트 평가 + 개선점 도출

# 명령문
너는 브랜드 디자인 디렉터이자 포스터 디자인 리뷰어야.

첨부한 초안 포스터 이미지를 분석하고,
브랜드 홍보 포스터로서 잘 된 점과 보완할 점을 리뷰해줘.

# 목적
현재 포스터가 브랜드 이미지, 제품 홍보, 행사 안내, 패키지 표현, 시각적 완성도 측면에서 적절한지 점검하는 것이 목적이야.

# 입력문

브랜드명:
[브랜드명 입력]

판매 상품:
[판매 상품 입력]

주요 타깃:
[주요 타깃 입력]

# 평가 기준
1. 브랜드명이 잘 보이는가?
2. 브랜드 콘셉트가 명확하게 느껴지는가?
3. 로고, 패키지, 굿즈, 포스터가 하나의 브랜드처럼 보이는가?
4. 제품이 매력적으로 보이는가?
5. 행사 정보가 잘 읽히는가?
6. 타깃 고객이 흥미를 느낄 만한가?
7. 포스터 안의 텍스트가 과하지 않은가?
8. 수정 없이 바로 사용할 수 있는 요소와 수정해야 할 요소는 무엇인가?
9. 같은 로고와 패키지를 유지한 채 배경만 바꿔 확장할 수 있는 구조인가?

# 제약 조건
- 단순히 “예쁘다”, “좋다”라고 말하지 말고 구체적인 이유를 작성해줘.
- 잘 된 점과 부족한 점을 구분해줘.
- 수정 방향은 실제 디자이너가 이해할 수 있게 작성해줘.
- 유지해야 할 요소와 변경해도 되는 요소를 분리해줘.

2. 브랜드 이름 후보 도출

활용 기법: 브레인스토밍 프롬프트 + 제약 조건 기반 네이밍 + 평가 기준 선제시

# 명령문

첨부한 초안 포스터와 브랜드 콘셉트를 바탕으로,
새로운 브랜드 이름 후보를 도출해줘.

# 목적
제품 패키지, 로고, 포스터, 굿즈로 확장하기 좋은 브랜드 이름 후보를 만드는 것이 목적이야.

# 입력문

[예: 수집하고 싶은 빵 브랜드, 선물용 패키지 브랜드, 팝업형 베이커리 브랜드]

주요 타깃:
[예: 20대 여성, 디저트 선물 소비자, 굿즈 수집을 좋아하는 고객]

# 네이밍 조건
1. 기억하기 쉬울 것
2. 발음하기 쉬울 것
3. 로고로 만들기 좋을 것
4. 한글과 영문 병기가 가능할 것
5. 패키지와 포스터에 넣었을 때 시각적으로 어울릴 것
6. 굿즈, 멤버십, 팝업 행사로 확장 가능할 것
7. 기존 유명 브랜드와 너무 비슷하지 않을 것

# 제약 조건
- 브랜드명 후보는 10개 제안해줘.
- 각 이름마다 의미를 설명해줘.
- 영문명과 한글명을 함께 제안해줘.
- 너무 추상적인 이름은 제외해줘.
- 상표 등록 가능성은 확정하지 말고 “상표 검색 필요”라고 표시해줘.
